ANIMACIONES DOM

Las animaciones de JavaScript se realizan mediante la programación de cambios graduales en el estilo de un elemento.Los cambios son llamados por un temporizador. Cuando el intervalo del temporizador es pequeño, la animación parece continua.

EVENTOS DOM

HTML DOM permite que JavaScript reaccione a eventos HTML, un JavaScript se puede ejecutar cuando ocurre un evento, como cuando un usuario hace clic en un elemento HTML. Para ejecutar código se debe agregar código JavaScript a un atributo de evento HTML

EVENTLISTENER DOM

  • El addEventListener()método adjunta un controlador de eventos al elemento especificado.
  • El addEventListener()método adjunta un controlador de eventos a un elemento sin sobrescribir los controladores de eventos existentes.
  • El addEventListener()método facilita el control de cómo reacciona el evento al burbujeo.
  • Al usar el addEventListener()método, el JavaScript se separa del marcado HTML, para una mejor legibilidad y le permite agregar detectores de eventos incluso cuando no controla el marcado HTML.

LISTA DE NODOS DOM

Un NodeListobjeto es una lista (colección) de nodos extraídos de un documento puesto que un NodeListobjeto es casi lo mismo que un HTMLCollectionobjeto.

Todos los navegadores devuelven un objeto NodeList para la propiedad childNodes ya que la mayoría de los navegadores devuelven un objeto NodeList para el método querySelectorAll().
